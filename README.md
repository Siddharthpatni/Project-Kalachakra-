## Core Research Question

> Can mathematically encoded astronomical and Vedic astrological representations improve probabilistic forecasting beyond conventional statistical baselines?

**H₀:** Features derived from Vedic astrological representations do not improve probabilistic forecasting beyond established statistical and ML baselines.

**H₁:** Certain mathematically encoded astronomical and Vedic feature representations provide statistically significant improvements over baseline models for specific prediction tasks.

## Quick Start

```bash
# Clone
git clone https://github.com/siddharthpatni/project-kalachakra.git
cd project-kalachakra

# Install (core only)
pip install -e ".[dev]"

# Install (all domains)
pip install -e ".[all,dev]"

# Run tests
make test

# Start API
make run-api

# Start dashboard
make run-dashboard
```

## Scientific Principles

This platform:

- **Never** claims deterministic prediction
- **Quantifies uncertainty** for every output
- **Separates** correlation from causation
- **Is reproducible** — fixed seeds, versioned data, captured environments
- **Is explainable** — SHAP, LIME, attention visualization
- **Is benchmarked** — against strong baselines (random, XGBoost, foundation models)
- **Supports hypothesis rejection** as readily as confirmation
